# Каталог библиотеки Свято-Троицкого Данилова монастыря Переславля
Набор утилит для формирования каталога монастырской библиотеки.

Общая схема добавления книг в библиотечный каталог:

* определение рубрики книги
* определение авторского знака по таблицам Хавкиной (часто в книгах авторский знак написан неправильно)
* присвоение очередного номера
* поиск книги в каталоге РГБ (rsl.ru) и импорт ее в Zotero через plugin
* уточнение данных в карточке (особенно с именами монашествующих путаница)
* внесение `Call number` в формате `А-1 Б33 01234` (рубрика, авторский знак, 5-значный инвентарный номер)
* нанесение от руки [в планах — печать] этикеток на обложку и корешок
* расстановка по полкам
* экспорт из `Zotero` в формат `Hayagriva`
* генерация каталога через `catalogue.typ`
